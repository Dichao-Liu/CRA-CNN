**CRA-CNN**

[matconvnet-1.0-beta25](https://github.com/vlfeat/matconvnet)

[mcnExtraLayers](https://github.com/albanie/mcnExtraLayers)

[mcnPyTorch](https://github.com/albanie/mcnPyTorch)

[autonn](https://github.com/vlfeat/autonn)

CRA-CNN
